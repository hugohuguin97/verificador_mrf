Información adicional para creación de archivos de datos

Como se indicó en la introducción, no tenemos la facultad de consultar directamente a la base de datos de imm-mage, por esta razón, es necesario exportar una serie de archivos csv para realizar las validaciones. A continuación se proporcionan las consultas que deben realizarse a mage con la finalidad de obtener los archivos necesarios para la ejecución de las validaciones.

Terminal.csv

SELECT Terminal.[$rdfId], ConnectivityNode.[$Path] FROM (Terminal WITH SUBTYPES LINK ON TerminalConnectedToConnectivityNode TO ConnectivityNode WITH SUBTYPES ) WHERE $path like 'Network*';
